MathJax FormInput Extension

A version of `forminput.js` for the [MathJax third party extension repository](https://github.com/mathjax/MathJax-third-party-extensions).

# About forminput

Provides HTML `` tag in MathJax math expression with `\FormInput` macro.

# Usage:

\FormInput[size][class][value]{name}

HTML `id` and `name` attributes given by value in required parameter, other attributes given by corresponding optional parameters.

*MUST* use either *native* MathML rendering or MathJax HTML/CSS rendering -- will *NOT* work if MathML rendering is handled by a plug-in, such as MathPlayer in Internet Explorer.
